# 동일성과 동등성

| 구분      | 비교 기준        | 비교 방법          | 예시            |
| ------- | ------------ | -------------- | ------------- |
| **동일성** | **객체의 주소값**  | `==` 연산자       | "완전 똑같은 객체냐?" |
| **동등성** | **객체의 값/내용** | `equals()` 메서드 | "내용만 같으면 되냐?" |


| 상황                     | 사용                |
| ---------------------- | ----------------- |
| 객체가 아예 같은 인스턴스인지 비교할 때 | `==` (동일성)        |
| 객체의 내용이 같은지만 비교하고 싶을 때 | `.equals()` (동등성) |
